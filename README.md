# Dinesh Interiors Prestige

A premium interior design portfolio website built with React, TypeScript, Vite, and Tailwind CSS.

## Project overview

This project showcases a curated interior design portfolio with animated sections, responsive layouts, and a polished visual experience for luxury residential spaces. It is designed to highlight completed projects, explain services, and capture leads through a contact experience.

## Featured projects

- Total projects displayed: 6
- Project categories: Kitchen, Bedroom, Living Room
- Example highlights:
  - Marble & Brass Kitchen — Bengaluru Residence
  - Carved Heritage Bedroom — Jaipur Villa
  - Arched Living Room — Mumbai Apartment

## Skills demonstrated

- React component design and state management
- TypeScript typing and safety
- Responsive layout with Tailwind CSS
- UI animation and interaction design
- Routing with React Router DOM
- Custom portfolio filtering and modal presentation
- Accessibility-friendly structure and focus handling

## Technical requirements

- Node.js 18 or newer
- npm, yarn, or bun package manager
- A modern browser for production preview

## Tech stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Radix UI primitives
- React Router DOM
- TanStack React Query
- Sonner toast notifications
- Recharts and React Day Picker (available utilities)

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Notes

The repository uses a `bun.lockb` lockfile, but it is compatible with standard npm workflows as well. Update the portfolio content or services pages to reflect your latest design case studies and client offerings.
